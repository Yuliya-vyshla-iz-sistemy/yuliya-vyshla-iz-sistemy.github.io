1. Общие положения

Настоящая Политика конфиденциальности определяет порядок обработки и защиты персональных данных пользователей сайта yuliya-vyshla-iz-sistemy.github.io и связанных сервисов (МедиаПлан Pro, Контент-Мастер), разработанных индивидуальным предпринимателем Пощевой Юлией Николаевной (ИНН 662330877581).

Используя наши сервисы, вы соглашаетесь с условиями настоящей Политики.

2. Какие данные мы собираем

При использовании приложений МедиаПлан Pro и Контент-Мастер мы можем собирать:

  • Адрес электронной почты (при регистрации)
  • Данные об использовании сервиса (количество запросов, выбранные функции)
  • Технические данные: тип браузера, IP-адрес, дата и время обращения
  • Данные, которые вы вводите самостоятельно (ниша, регион, тексты для обработки)

Главный лендинг (эта страница) не содержит форм сбора данных. Cookies используются только для технической работы сайта и аналитики.

3. Цели обработки данных

  • Предоставление доступа к функциям сервисов
  • Обработка платежей через ЮKassa
  • Техническая поддержка пользователей
  • Улучшение качества сервисов
  • Отправка уведомлений об изменениях в сервисах (с вашего согласия)

4. Хранение и защита данных

Данные хранятся в защищённой базе данных Supabase (supabase.com). Платежи обрабатываются через ЮKassa (yookassa.ru) — мы не храним данные банковских карт.

Мы принимаем технические и организационные меры для защиты ваших данных от несанкционированного доступа.

5. Передача данных третьим лицам

Мы не продаём и не передаём ваши персональные данные третьим лицам, за исключением:

  • Supabase — для хранения данных аккаунта
  • ЮKassa — для обработки платежей
  • ИИ-провайдеров (DeepSeek, Gemini и др.) — только тексты запросов, без персональных данных
